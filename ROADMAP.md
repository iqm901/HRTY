# HRTY Roadmap

- Add new ideas under the appropriate priority section
- Include screenshots in `docs/images/` and reference them here

---

## How to Add Screenshots

1. Save your screenshot or mockup to `docs/images/`
2. Reference it in this file using: `![Description](docs/images/your-image.png)`

Example:
```markdown
![New Dashboard Design](docs/images/dashboard-mockup.png)
```
